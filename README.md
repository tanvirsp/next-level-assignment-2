# DevPulse

#### Develped by Mir Raisul Tanvir

This is simple backend assignment proejct for NEXT LAVEL WEB DEVELOPER

### Technology Used:

- Express JS
- Typescript
- PostgraSQL
- JWT [ jsonwebtoken ]

### Features

- Role base authorization
- Two type of role ( contributor and maintainer)
- User can create account
- Create new issue
- Only contributor can update won issue
- Maintainer can update any issue
- Maintainer can delete any issue
- Maintainer can change issue workflow status independently
- Auth verify using jwt token
